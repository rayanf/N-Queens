# N-Queens
Solving N-Queens problem by implementing Memetic Algorithm.

## Usage
Call training function from functions.py.
```python
training(n,m,selection_rate, cross_rate, mutation_rate,temp,cooling,k_near,itteration=100,repetition=2)                    
```
